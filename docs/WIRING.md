# 🔌 Wiring Guide — التوصيل التفصيلي

> ⚠ **تحذير**: دوائر الـ induction تعمل بتيارات عالية (300A+) وفولتية قاتلة. ارتدِ قفازات عازلة، اعزل كل التوصيلات، وثبّت قواطع مناسبة.

## 3) الـ Resonant Tank

تختلف قيم L و C حسب المعادن المستهدفة. جدول عام:

| المعدن   | التردد النموذجي | L (µH) | C (µF)  |
|----------|-----------------|--------|---------|
| ألمنيوم  | 30 kHz          | 8      | 3.5     |
| نحاس     | 80 kHz          | 3      | 1.3     |
| حديد     | 45 kHz          | 6      | 2.1     |
| ذهب      | 120 kHz         | 2      | 0.9     |
| فضة      | 100 kHz         | 2.5    | 1.0     |

صيغة الرنين:
```
f = 1 / (2π √(L × C))
```

تأكد أن `f_resonance` قريبة من تردد التشغيل الذي تختاره في الواجهة (نسبة 1:1.2 مثلاً).

## 4) حساس التيار (ACS712-30A)

```
VCC ─┬─ 3V3
     │
     ┴  0.1µF (decoupling)
     │
     ┬  OUT ─────────► ESP32 GPIO 35 (ADC)
     │
     ┴  (مكثف 100nF على الخرج لتنعيم)
     │
    GND
```

> **تحذير**: ACS712-30A يعطي 66mV لكل أمبير. تأكد أن التيار لا يتجاوز 30A RMS، وإلا يحترق.

---

## 5) مقسم جهد Vbus

```
Vbus + ─── R1 (100k, 1W) ───┬─── ESP32 GPIO 32 (ADC)
                              │
                              ┴  R2 (10k, 0.25W)
                              │
                             GND
```

نسبة = 100k / (100k + 10k) = 1/11.
`Vbus_actual = Vadc × 11`

> استخدم مقاومات عالية (100k) لتقليل التيار المهدور. ADC لا يستهلك تياراً يُذكر.

## 6) حساس الحرارة MAX6675 + Type-K

```
MAX6675        ESP32
─────────────────────
VCC      ◄───  3V3
GND      ◄───  GND
SCK      ◄───  GPIO 19  (SPI SCK، مشترك مع SD)
SO/MISO  ───►  GPIO 18  (SPI MISO، مشترك مع SD)
CS       ◄───  GPIO 5   (low to read)
```

> تأكد أن السلك الحراري بعيد عن الـ coil (50cm+ على الأقل) لتجنب التداخل الكهرومغناطيسي.

---

## 7) حساس تدفق المياه YF-S401

```
VCC (RED)   ────  5V (منظم منفصل، ليس 3V3)
GND (BLACK) ────  GND
YEL (DATA)  ────  ESP32 GPIO 4   (+ 10k pull-up إلى 3V3)
```

العامل: **450 نبضة / لتر** (5.5Hz لكل L/min).

> الـ 5V يمكن أن تأتي من محول 5V/3A منفصل، أو منظم LM7805 على دخل USB.

## 8) NTC (10k B3950) × 2

نفس الدائرة لـ IGBT و coolant:

```
3V3 ──┬── R_series (10k) ──┬── ESP32 GPIO (ADC)
       │                    │
       │                    ┴  NTC 10k
       │                    │
      GND                  GND
```

**معادلة Steinhart-Hart (مبسّطة Beta):**

```
1/T = 1/T₀ + (1/β) × ln(R_ntc / R₀)

T بالكلفن، β = 3950، R₀ = 10k @ 25°C
```

## 9) الأزرار والـ LED

```
        3V3
         │
         ┴  10k pull-up
         │
GPIO 2 ──┬──── BTN_START ──── GND
         │
        LED + 220Ω ──── GPIO 2 ──── GND
        (built-in LED على معظم اللوحات)


GPIO 0 ──┬──── BTN_STOP/E-STOP ──── GND
         │
         ┴  10k pull-up داخلي
```

> GPIO 0 هو BOOT — للاستخدام الدائم، ضع **زر خارجي pull-up**، وافصل الـ USB بعد التحميل.

## 10) الـ Relay الرئيسي (230V contactor)

```
ESP32 GPIO 23 ──┬── 1k ──┬── NPN base (BC547)
                 │        │
                 │        C
                 │        │
                 │       Relay coil
                 │        │
                GND       E
                          │
                         GND

+ ضع ديود 1N4007 عكس قطبية على الـ coil
```

الـ relay يجب أن يكون **30A+** بملف **5V أو 12V** حسب تغذيتك.

---

## 11) التوصيل بالـ Coil (الملف الحثي)

- استخدم أنبوب نحاسي بقطر داخلي 25-40mm وارتفاع 50-80mm.
- عدد اللفات: 8-12 لفة من سلك نحاسي 6mm².
- ضع البوتقة (crucible) في المنتصف بحيث تكون باردة من الجانب السفلي.
- وصّل نهاية الملف بمكثف الرنين، والأخرى بالأرضي للمحوّل.

---

## 12) التأريض (Grounding) والسلامة

- **يجب** أن يكون كل من:
  - جسم العلبة المعدنية
  - الأرضي للـ coil
  - الأرضي للـ mains

  مرتبطاً بنقطة أرضي مشتركة، ومتصلاً بالأرضي الحقيقي للمبنى.

- ضع **RCD/GFCI** على مدخل التيار الكهربائي.

- ابدأ دائماً على **dummy load** (مكثف + مقاومة 50Ω 100W) قبل توصيل الـ coil.

---

## 13) الـ enclosure (العلبة)

- استخدم علبة **معدنية مؤرضة**.
- اجعل الـ coil داخل تجويف منفصل.
- ضع مراوح تبريد 80mm على IGBT.
- ضع شاشة أمان بليكسي سميك بين المستخدم والـ coil.

---

## 14) الاختبار الأول

1. ارفع البرنامج بدون توصيل الـ coil.
2. افتح الواجهة، راجع الـ status.
3. فعّل PWM مع dummy load، تحقق من شكل الموجة على oscilloscope.
4. تحقق من قراءة Vbus, Current, Temperatures.
5. وصّل الـ coil بدون معدن، شغّل لـ 5 ثواني، تحقق من التيار (< 15A).
6. ضع قطعة ألمنيوم صغيرة، راقب الحرارة حتى تنصهر.

> **لا تترك الجهاز بدون مراقبة أبداً!**
